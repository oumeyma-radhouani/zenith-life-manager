> A modern, full-stack task management platform designed to turn daily habits into an engaging RPG experience.

## ( •̀ ω •́ )✧ Project Overview
Zenith is a productivity engine that leverages core gamification mechanics—such as experience points (XP), leveling systems, and a token-based reward economy—to motivate users and build momentum in their daily lives. 

## (ﾉ◕ヮ◕)ﾉ*:･ﾟ✧ Key Features
* **Dynamic Task Management:** Create, track, and complete daily habits with varying difficulty multipliers.
* **RPG Mechanics:** Earn XP and level up by maintaining productivity streaks.
* **Token Economy:** Accumulate tokens from completed tasks to spend on custom, user-defined rewards.
* **Modern SaaS Architecture:** Built with decoupled frontend and backend services, featuring instant API documentation and secure, cloud-based data storage.

## (✿◠‿◠) Tech Stack
This project is built using a modern, market-leading technology stack:
* **Frontend:** Next.js (React) & Tailwind CSS
* **Backend:** FastAPI (Python)
* **Database & Authentication:** Supabase (PostgreSQL)
* **Infrastructure:** Docker & Vercel
